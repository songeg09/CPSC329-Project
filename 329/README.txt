To run the program

1. open up your prompt

2. change directory to src in this current file.

3. run the following lines

	javac -encoding utf-8 Game.java

	javac -encoding utf-8 JeopardyGame.java

	javac -encoding utf-8 KeyListener.java

	javac -encoding utf-8 Main.java

	javac -encoding utf-8 Quiz.java

4. finally you can run the game by the following line

	java Main
